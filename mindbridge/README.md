Mind Bridge offers a safe and supportive space for individuals experiencing anxiety, depression, stress, grief, loneliness, and more. ## Installation

1. Clone the repository
2. Copy `.env.example` to `.env` and configure your environment variables
3. Install dependencies:
   ```bash
   npm run install-all
   ```

4. Start development server:
   ```bash
   npm run dev
   ```
